# gte

"Greater than or equal to" ( >= ) takes two arguments and returns `true` if the first argument is greater than or equal to the second argument; otherwise returns `false`.

# API
`(a, b) => a >= b`

# Example

```
gte(5, 3) === true
gte(5, 5) === true
gte(5, 7) === false
```
